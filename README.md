# TP-SYSTEMES
TP SYSTEMES

LE BUT -> Faire un contrôle avec des lumières afin de pouvoir changer leurs couleurs grâce au protocole DMX512.

NOTRE MAGINIFIQUE TP SYSTEME AVEC ELOISE, TOM ET LUCAS.

## 1) ORGANISATION DU CODE

* ./QtDMX
  
    *QtDMX.cpp* -> fichier de déclaration de la classe QtDMX
  
    *QtDMX.h* -> fichier d'entête de la classe QtDMX
  
    *DasHard2006.dll* -> Fichier DLL de la bibliothéque DasHard

    *QtDMX.qrc* -> Fichier QRC du projet (pour gérer l'interface QT)

    *QtDMX.pro* -> Fichier PRO du projet (pour gérer aussi l'interface QT)

    *PilotageLumiere.h* -> fichier d'entête de la classe PilotageLumiere

    *PilotageLumiere.cpp* -> fichier de déclaration de la classe QtDMX

    *PilotageLumiere.cpp* -> fichier de déclaration de la classe QtDMX

    *main.cpp* -> fichier principal
  
    *QtDMX.sln*, fichier de la solution du projet qui  permet son fonctionnement
  
    *QtDMX.vcxproj.filters* -> fichier de configuration de la solution
  
    *QtDMX.vcxproj* -> fichier de configuration de la solution
  
    *QtDMX.vcxproj.user* -> fichier de configuration de la solution
